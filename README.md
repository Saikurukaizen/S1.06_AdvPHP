Carpetas de los ejercicios del Tema 6 - PHP Avanzado

- PHP 8.1.10
- IDE: Visual Studio Code
- Git & GitHub

## 📋 Requisitos
- Servidor local tipo Laragon, XAMPP / MAMP / LAMP para la compilación de PHP.
